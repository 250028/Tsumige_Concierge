積みゲー・コンシェルジュ
### サブタイトル
あなたの罪悪感をワクワクに変える、積みゲー消化応援サポーター
## テーマ / ジャンル
ゲーミングライフログ / エンタメ / AIチャット(仮)

## 誰のためのアプリか（想定ユーザー）
- セールや新作発表のたびにゲームを買い足し、未プレイの「積みゲー」が山積みになっているゲーマー

- 次に何のゲームを遊ぶべきか、選択肢が多すぎて迷ってしまう人

- ゲームを途中で放置してしまい、再開するキッカケを失っている人

## 何を解決するか（困りごと）

- 積みゲーが増えるたびに「お金と時間を無駄にしている」という罪悪感がある
- 「せっかくまとまった時間ができたのに、何を遊ぶか迷っているうちに休みが終わった」というタイムロス
- 久しぶりに起動するゲームのキー配置や基本操作を忘れてしまい、思い出すまでにストレスがかかる
- ストーリー進行度や前回のあらすじを忘れてしまい、ゲームの世界観への没入感が薄れて再開を諦めてしまう

## どう解決するか（このアプリでできること）

- 自分の積みゲーリストから、今の気分や所有時間に合わせた「最適な1本」をAIが選んでくれる
- ゲームのキャラクターや「ゲームの精霊」のようなAIが、チャットで「そろそろ世界を救いに行こう！」と楽しくモチベーションを上げてくれる
- 放置していたゲームの「前回までのあらすじ」やAIに聞くことで、スムーズにプレイを再開できる
- プレイを再開したいゲームの「プラットフォーム（Switch/PS5/PCなど）」を指定するだけで、AIがそのゲームの基本操作方法やボタン配置を分かりやすく一覧で教えてくれる

## 主な機能（やりたいことを箇条書きで）

- 積みゲーマイリスト機能： タイトル、ジャンル、購入日、現在の進捗（未開封・序盤で放置など）を登録・管理
- AIコンシェルジュ（チャット）： 「今から2時間暇」「とにかくスカッとしたい」などとチャットすると、リストからおすすめを提案
- おまかせランダムセレクト： 迷いすぎて選べない時、AIが演出つきで今日の1本を決定
- あらすじ・モチベーター機能： 「〇〇（ゲーム名）を再開したいけど、どこまでやったか忘れた」と打つと、一般的なストーリー進行度に合わせてAIが要約・解説してくれる
- 対象ゲームと「ハードウェア（PS5、Switchなど）」を選択し、「操作方法」ボタンを押すと、「移動：Lスティック」「攻撃：□ボタン」といった主要な操作一覧をAIが即座に生成して画面に表示する。

## AIで実現したいこと（どんなAI機能を、どう使うか）

- ゲームデータに基づいたマッチング： LLM（大規模言語モデル）にゲームの一般的なクリア時間やジャンルの特徴を学習させ、ユーザーの条件（例：15分で終わる、死にゲー以外など）に合わせた高度なフィルタリングを行う
- ペルソナ（キャラクター設定）チャット： AIの話し方を「厳格な執事」「ゲーマー仲間」「ゲームのナビゲーター妖精」などに切り替え、プレイヤーを楽しく鼓舞する
- ゲーム情報の補完： ユーザーがタイトルを入力するだけで、AIがそのゲームのジャンルや魅力を自動で認識し、リストのタグ付けをサポートする

## 実装の難しさ（予想）

- ゲームの基本情報（ジャンルやクリア想定時間など）を、外部のゲームデータベースAPIから引っ張ってくるか、すべてAI（LLM）の知識に頼るかの設計の割り切り
- AIが「嘘のゲーム情報（ハルシネーション）」を答えないよう、プロンプトで厳密に制御する必要がある
- ユーザーごとの積みゲーリストを保持するための、データベース（DB）の設計

## 自分だけの独自要素
単なる「消化の催促」ではなく、ゲームをプレイすること自体をゲーム化（ゲーミフィケーション）する点。積みゲーを1本クリアするごとにアプリ内でポイントが貯まり、AIの衣装が変わったり、自分の「積みゲー城」が崩れて綺麗になっていくような視覚的達成感を取り入れる。

## 使う技術スタックと、その理由

- フロント：Next.js/CSS(Tailwind) React ベースでページ遷移・API呼び出しを一元管理できるため
- バックエンド：Next.js API Routes フロントと同一プロジェクトで管理でき、初学者にとってシンプルな構成になるため
- DB：MySQL 授業で学んだ技術を活かせる・リレーショナル設計に適しているため
- AI：Gemini API 画像分類・テキスト生成の両方に対応でき、無料枠が充実しているため

## AIツールの活用（使った場合：ツール名と使い方）

- Google AI Studio：Gemini APIのプロンプト（ゲームの操作方法やあらすじを綺麗に出力させるための命令文）の調整・テストに使用。
- Cloaude Code: コーディング
